# QuickMathSolver

A simple command-line math calculator for quick calculations.

## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

Enter math expressions like:
- 2 + 3
- 5 * 8
- 10 / 2
- (5 + 3) * 2

Type "exit" to quit.

## Features

- Basic arithmetic operations
- Interactive command line interface
- Error handling for invalid expressions
